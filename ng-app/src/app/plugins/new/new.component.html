Plugin license

You are going to create a plugin that is supposed to run on the official Twasi-Core instance. For that it is necessary to license your plugin with the TOPL (Twasi Official Plugin License) that you can see below.

Please read it carefully and then confirm that you want to include it in your plugin.

You are not allowed to combine the TOPL with any other license.
